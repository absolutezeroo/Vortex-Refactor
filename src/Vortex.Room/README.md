## SDK

`Microsoft.NET.Sdk` — References `GodotSharp`/`Godot.SourceGenerators` 4.6.1.

## Directory Structure (empty scaffolds)

```
src/
  data/           Room data models
  events/         Room events
  exceptions/     Room exceptions
  messages/       Room protocol messages
  object/         Room objects (furniture, avatars, pets)
  renderer/       Room rendering pipeline
  utils/          Room utilities
```

All directories are empty, ready for AS3 porting.
